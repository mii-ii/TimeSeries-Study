# consumption_forecast(초기 모델) 성능 분석

### 1. R² 점수가 음수(-0.8340)인 이유
보통 R^2 (결정계수)는 0에서 1 사이의 값을 가지지만, 본 실험에서는 -0.8340이라는 음수 값이 도출되었다. 이는 현재의 단순 모델이 평균값으로 예측하는 것보다 성능이 떨어짐을 뜻하며, 해당 기간의 소비 패턴을 전혀 맞히지 못하고 있다.

### 2. 그래프 시각적 분석
* 초록색 점선 (Prediction) :
    * 모델은 과거 데이터(1970~2007년)만 보고 학습하였다.
    * 과거에는 큰 폭락 없이 일정한 패턴이 지속되었기에, 모델은 "앞으로도 소비가 예전처럼 0.8~1.0 수준을 웃돌 것"이라고 낙관적으로 예측했다.
* 주황색 실선 (Actual) :
    * 실제 데이터는 2008년 무렵 우하향한다.
    * 이 시기는 2008년 세계 금융 위기로, 경제가 붕괴되며 소비 심리가 급격히 위축된 시점이다.
* 본 모델은 '과거의 평화로운 패턴'만 학습했기 때문에, 구조적 변화인 경제 위기 상황을 전혀 예측하지 못하였다.

### 3. 결론 및 해결책
본 모델은 "단변량(Univariate) 시계열 예측의 한계"를 보여준다.
* 소비(Consumption)는 단순히 시간의 흐름(Autoregression)만으로는 예측할 수 없다. 소비 지출은 소득(Income)이나 실업률(Unemployment) 같은 외부 거시경제 요인의 영향을 크게 받으므로, 이를 모델에 변수로 추가해야 예측 정확도를 높일 수 있다.
